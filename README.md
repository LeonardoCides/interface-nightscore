# 📅 EventFinder - Interface de Divulgação de Eventos

> Uma interface web moderna, leve e responsiva para a visualização e busca de eventos locais.

---

## ✨ Funcionalidades

* 🔍 **Filtro de Eventos:** Sistema de busca dinâmica por nome ou categoria.
* 📱 **Design Responsivo:** Layout adaptável para dispositivos móveis e desktops.
* 🖼️ **Galeria de Cards:** Exibição clara com imagens, datas e descrições dos eventos.
* 🖱️ **Interatividade:** Efeitos de hover e transições suaves com CSS e JS.
* 🎫 **Modal de Inscrição:** Interface para detalhes do evento e confirmação de presença.

---

## 🛠 Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando tecnologias nativas para garantir máxima performance:

* **HTML5:** Estruturação semântica de todo o conteúdo.
* **CSS3:** Estilização avançada com Flexbox, Grid e animações.
* **JavaScript (ES6+):** Manipulação do DOM e lógica de filtragem de eventos.
